Down the Line

I feel very privileged to have the opportunity to learn to code, and as a result I want to give back in areas that I am passionate about. Specifically, I would like to use this opportunity to give a voice to those who aren't able to speak for themselves (animals).

In addition to this, I would like to encourage others coming into the tech industry by either mentoring women, or potentially volunteering in a developing country to teach kids how to code.

Process

I spent some time in the past 2 years doing some self-study in software development. I decided to start with using the Angular framework along with the inevitable HTML, CSS & JavaScript. This site you’re looking at was build from the ground up in Angular 8 as a Single Page application. I know it’s not perfect, for example it doesn’t fit perfectly on all device sizes, and I still need to get the service workers functional to make it a progressive web app.

I think this app does showcase some of my learnings, for example:

  • Using Angular to create an app
  • Using HTML to build the page components
  • Building a menu with special formatting (I used HTML tables)
  • Using CSS to style what you see
  • Using JavaScript to call function that changes the content of the page
